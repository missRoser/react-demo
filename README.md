
react制作的新闻列表页，功能主要包括:
1、展示新闻列表页，根据不同的按钮排序
2、搜索关键字,利用fetch搜索并展示到页面。
3、本地存储，搜索过的关键字存储和内容利用缓存存储在本地。
4、利用bootstrap,font-awesome来实现样式和字体图标。

## 操作
git clone https://github.com/missRoser/react-demo.git
cd react-demo
npm init --yes
npm install
npm run start
